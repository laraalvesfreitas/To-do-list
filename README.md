# Projeto To-do List

Este é um projeto de lista de tarefas desenvolvido com HTML, CSS e JavaScript, com o objetivo de ajudar na organização de atividades do dia a dia.

A aplicação permite adicionar tarefas e acompanhar o que precisa ser feito, sendo uma ferramenta simples para melhorar a produtividade pessoal.

## Funcionalidades
* Adicionar novas tarefas
* Listar tarefas cadastradas
* Remover tarefas
* Persistência de dados (LocalStorage)
  
## Objetivo do projeto

Este projeto foi criado com foco em prática de desenvolvimento front-end, especialmente manipulação de DOM com JavaScript puro.

## Tecnologias utilizadas
* HTML
* CSS
* JavaScript
